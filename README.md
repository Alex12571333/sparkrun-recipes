# Alex12571333's sparkrun recipes

Recipes for [sparkrun](https://github.com/spark-arena/sparkrun) optimized for **NVIDIA DGX Spark (GB10)**.

## Usage

```bash
sparkrun registry add https://github.com/Alex12571333/sparkrun-recipes
sparkrun run gemma-4-12b-it-nvfp4-w4a16-mtp
```

## Recipes

- `gemma-4-12b-it-nvfp4-w4a16-mtp` — Gemma 4 12B with NVFP4 weight-only + MTP, multimodal intact, ~44 tok/s on Spark
